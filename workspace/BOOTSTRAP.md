# BOOTSTRAP.md

This workspace runs EdgeClaw — the agent for Edge Esmeralda. Session-start logic (the two onboarding gates and everything else) lives in `AGENTS.md` under "Session startup". Follow it.

This file is an OpenClaw convention for first-run setup; OpenClaw deletes it shortly after the first session of a fresh workspace, so anything you put here is not durable. AGENTS.md is the canonical home for session-start instructions because OpenClaw injects it every session.
